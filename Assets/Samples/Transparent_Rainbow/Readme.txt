# 概要
VRChatのワールド作成に使用することを想定したシェーダーです。
適用したオブジェクトがピカピカと虹色に光ります。
マテリアル毎にひかり具合を設定することができます。
Speed : 値を高くすれば光るスピードが速くなります。
Value : 値が高ければ高いほど明るく光ります。
Saturation : 値が低ければ低いほど白っぽくなり、高ければ高いほど透明に近くなります。

Amplify Shader Editor を使用して作成したので、そちらを所有している方は中身を開いて簡単に改変できます。

# 内容物
unitypackageの中に以下が入っています。
- transparent_rainbow.shader : shader本体です。
- hanawa_example : 使用例として用意した花輪です。
- hanawa : 花輪の素材が入っています。
- example_materials : 花輪で使用しているマテリアルやその他値を設定したマテリアルが入っています。
- Readme.txt : これ。
